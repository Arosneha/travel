## 🚀 Getting Started

### Prerequisites

- **Node.js:** 14.0 or higher
- **npm:** 6.0 or higher (comes with Node.js)
- **Git:** Latest version
- **Code Editor:** VS Code recommended

### 💻 Local Setup

#### Step 1: Clone the Repository
```bash
git clone https://github.com/Arosneha/travel.git
cd travel
```

#### Step 2: Install Dependencies
```bash
npm install
```

#### Step 3: Start Development Server
```bash
npm start
```

The app will open automatically at **`http://localhost:3000`** 🎉

## 🌐 Deployment

### Deployed on Netlify

Your app is live at: **[https://holiday-fun.netlify.app](https://holiday-fun.netlify.app)** 🚀

### How to Deploy to Netlify

#### Method 1: Using Netlify UI (Easiest)

1. **Build Your App**
   ```bash
   npm run build
   ```

2. **Go to Netlify.com**
   - Visit https://app.netlify.com

3. **Drag & Drop**
   - Drag the `build` folder to Netlify
   - Your site is now live!

#### Method 2: Connect GitHub (Recommended)

1. **Push Code to GitHub**
   ```bash
   git add .
   git commit -m "Holiday Fun - Ready to deploy"
   git push origin main
   ```

2. **Go to Netlify.com**
   - Click "New Site from Git"
   - Connect GitHub
   - Select repository
   - Build command: `npm run build`
   - Publish directory: `build`
   - Deploy!

3. **Auto Deploy**
   - Every push to main branch auto-deploys!

#### Method 3: Using Netlify CLI

```bash
# Install Netlify CLI
npm install -g netlify-cli

# Build the project
npm run build

# Deploy
netlify deploy --prod --dir=build
```

## 🆘 Troubleshooting

### Issue: Port 3000 Already in Use
```bash
# On Windows:
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# On Mac/Linux:
lsof -ti:3000 | xargs kill -9

# Or use different port:
PORT=3001 npm start
```

### Issue: Dependencies Not Installing
```bash
# Clear npm cache
npm cache clean --force

# Delete node_modules and package-lock.json
rm -rf node_modules package-lock.json

# Reinstall
npm install
```

### Issue: Build Fails
```bash
# Clear cache and rebuild
npm run build -- --reset-cache

# Or clean build
npm cache clean --force && npm install && npm run build
```

### Issue: Images Not Loading
```javascript
// Check image paths
// Use relative paths: ./images/destination.jpg
// Or absolute paths from public folder: /images/destination.jpg
```

### Issue: Netlify Deploy Fails
```bash
# Make sure you have:
1. package.json with all dependencies
2. build folder created successfully
3. npm run build works locally
4. .gitignore excludes node_modules
5. Netlify has correct build command
```
